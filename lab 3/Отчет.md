Отчет по лабороторной работе №3  
Программа для составления меню из чая и кофе  
Параметры напитков: Тип, производитель, сахар, начальная цена, итоговая цена    
Итоговая цена высчитывается из начальной цены + количество сахара * 3  
1) Ввод реализован при помощи lazy list  
2) Обработаны ошибки на неправильный формат
3) Валидируются значения параметров  
Приемлимые значения:  
Тип: Tea | Coffee  
Производитель: Любой  
Сахар: 1-100  
Начальная цена: 1-5000
